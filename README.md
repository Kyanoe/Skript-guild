# Skript-guild

Tested Minecraft Version: 1.16.x

List Commands:

- /guild help <page>
- /guild create <guild>
- /guild delete
- /guild invite <player>
- /guild accept <guild>
- /guild decline <guild>
- /guild join <guild>
- /guild name <new name>
- /guild description <new guild description>
- /guild warInvite <guild>
- /guild warAccept <guild>
- /guild warDeny <guild>
